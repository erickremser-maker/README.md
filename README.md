# Zentrales Projektportal – React/TypeScript-Prototyp

Klickbarer Oberflächenprototyp auf Basis des fachlichen Anforderungspakets im Ordner `docs`. Die Anwendung nutzt ausschließlich lokale JSON-Beispieldaten und enthält keine echte Datenbank, keine echte Anmeldung und keine produktive Backend-Integration.

## Startanleitung

```bash
npm install
npm run dev
```

Danach die von Vite ausgegebene lokale URL im Browser öffnen.

## Weitere Befehle

```bash
npm run build
npm run preview
```

## Fachlicher Umfang

Der Prototyp enthält Portfolio-Startseite, Projektakte, Rollenmatrix, Phasenübersicht, Planprüfung, BVB-Freigabe, Terminübersicht, Terminverschiebungsformular, Risikoübersicht, Budgetansicht, Stundenansicht, Dokumentenbereich, Entscheidungslog und Wissensdatenbank.

## Daten und Datenschutz

Alle Inhalte liegen in `src/data/portalData.json`. Die Beispieldaten sind anonymisiert und verwenden Platzhalter für Personen, Rollen, Standorte und Anlagen.
